## ✨ Features

### 📊 Job Application Management
- Track job applications in one place
- Organize applications by status (Applied, Interview, Offer, Rejected)
- Add company details, role, and notes

### 📌 Kanban Board
- Drag & drop job applications between stages
- Visual workflow for job search progress

### 🤖 AI Features
- Resume improvement suggestions
- Job description matching insights
- Cover letter generation (optional feature)

### 📈 Analytics Dashboard
- Application statistics
- Interview conversion rate
- Progress tracking

### 🔐 Authentication
- Secure login/signup system
- JWT-based authentication
- Optional OAuth (Google login)
